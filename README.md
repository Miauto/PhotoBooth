# PhotoBooth
Conception de pieces et assemblage pour un Photobooth.
Rassemblement d'idées et des schémas pour la conception.

# Contenue
* Raspberry pi 5 - 8Go ram
* Carte Nvme - https://wiki.geekworm.com/X1001
* NVME WD blue - 1 To
* Ecran 7" (Bug du tactile à l'allumage, un bouton permet de couper et rallumer l'écran)
* Camera pi V3
* LED RING 16 led - piloté par ESP8266 sous WLED - connecté en USB
* Imprimante Canon CP1500 - Bug (bloque en fin d'impression)
* Eclairage LED
* Multi port USB en interne
* Bouton de déclenchement en Bluetooth
* embase d'enceinte 35mm (trou pour pied d'enceinte)
* Multi-prise de 3 prises
